# Apliación de Horoscopo en Android nativo.


Herramientas utilizadas:
- Arquitectura MVVM y clean code
- Fragments
- Navigation Component
- Gradle KTS
- Inyección de dependencias
- StateFlow y corrutinas
- RecyclerView
- Retrofit, interceptors y mappers
- Intents
- Camera X
- Animaciones
- UnitTest y UITest
